Report

Reflective Introduction

Embarking on the journey of web development this term has been an enlightening and transformative experience. From learning the basics of HTML and CSS to designing and debugging each step presented unique challenges and rewards. This report captures my journey, focusing on both the technical development and design aspect of my website, and includes a video demonstration link and validation screenshots.

Technical Development

Starting with HTML and CSS, I learned to structure web pages and style them effectively. Creating my first web page was a thrilling experience, and understanding the importance of semantic HTML tags like header, nav, article and footer set the foundation for accessible and well structured web content.

Design Development

Desgning the UI was a creative highlight. My goal was to create an aesthetically pleasing and user-friendly site. Consistency across pages was key, achieved through a fixed navigaion menu and a cohesive color scheme.

Design Decisions

I selected "poppins" for body text due to its readability and modern feel for heading to add contrast and elegance. Font were sourced from google fonts.A clean layout with a fixed top navigation bar ensured ease of access. The site was designed to be responsive, with a hamburger menu for mobile devices, ensuring functionality across all screen sizes.
